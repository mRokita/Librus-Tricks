Sesja z Synergią
******************

Co robi obiekt sesji dlaczego jest taki ważny?
===============================================
Librus-tricks spaja wiele komponentów w jedną całość.
Utworzenie sesji pozwala na wygodne wykorzystywanie różnych userów lub mechanizmów cache.

Modularność takiego rozwiązania pozwala na szybkie tworzenie nowych sesji. Można to trochę porównać do rozwiązania znanego z Project Treble.
